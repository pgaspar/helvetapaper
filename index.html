Helvetapaper

a minimalistic theme for Instapaper

Helvetapaper is a minimalistic theme for Instapaper. Helvetapaper gives this wonderful web application a cleaner, more objective look, focused and reduced on functionality which is—in my opinion—the essential minimum. See Fig.1 for a screenshot.

Instapaper is a simple tool to save web pages for reading later by Marco Arment.

Helvetapaper is highly inspired by all the Helvetica userscripts out there, but most of all by the Documentary Film Helvetica by Gary Hustwit from 2007.

Helvetapaper hides some features of Instapaper. This was a deliberate decision, because I currently do not use them at all. If I will find them useful one day, I will of course unhide them. You may also drop me an Issue in Github, so I will consider it on future changes. You may also fork this userscript and fix it for your personal needs. On the other hand, Helvetapaper adds some new features which are vital for my daily usage of Instapaper. Currentyl these are:

  • Show number of links.
  • Flag new links (new links from within the last three days).

Helvetapaper Identity

Helvetapaper is based on Helvetica typeface.

Helvetica is a sans-serif typeface by Max Miedinger and Eduard Hoffmann released in 1957.

Usage

In order to theme your personal Instapaper with Helvetapaper, you need to have Greasemonkey installed. Install Greasemonkey here.

Installing

Install Helvetapaper userscript

Uninstalling

To uninstall a Greasemonkey script from your browser, go to the Tools menu, select Greasemonkey, then Manage User Scripts, highlight the script and click Uninstall.
