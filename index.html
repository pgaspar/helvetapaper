Helvetipaper

a minimalistic theme for Instapaper

Helvetipaper is a minimalistic theme for Instapaper. Helvetipaper is highly inspired by all the Helvetica userscripts and by the Documentary Film Helvetica by Gary Hustwit.

Instapaper is a simple tool to save web pages for reading later by Marco Arment.

Helvetipaper is based on Helvetica typeface. To use Helvetipaper you need to have Greasemonkey installed. Install Greasemonkey here.

Helvetica is a sans-serif typeface released in 1957.

This userscript does hide some features of Instapaper. Fork this userscript and fix this for your personal needs. Helvetipaper adds some features which are vital for my daily usage of Instapaper:

  • Show number of links.
  • Flag new links (new links from within the last three days).

Usage

Installing

Uninstalling

To uninstall a Greasemonkey script from your browser, go to the Tools menu, select Greasemonkey, then Manage User Scripts, highlight the script and click Uninstall.
