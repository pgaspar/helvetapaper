Helvetapaper

a minimalistic theme for Instapaper

Helvetapaper is a minimalistic theme for Instapaper. Helvetapaper gives this terrific web application a cleaner, more objective look, focused and reduced on functionality which is—in my opinion—an essential minimum. See Fig.1 for a screenshot.

Instapaper is a simple tool to save web pages for reading later by Marco Arment.

Helvetapaper is highly inspired by all the Helvetica userscripts out there, but most of all by the Documentary Film Helvetica by Gary Hustwit from 2007.

Helvetapaper hides some features of Instapaper. This was a deliberate decision. I currently do not use all features of Instapaper. If you are missing features which are important for you, drop me a Github issue or fork Helvetapaper and customize it for your personal needs.

On the other hand, Helvetapaper adds some new features which are vital for my daily usage of Instapaper. The following enhancements are currently implemented:

  • page-counter flag on each top-category link (Unread, Starred, Archive).
  • new-page flag on saved pages from within the last three days.

Helvetapaper Identity

Helvetapaper is based on the Helvetica typeface. Considering web fonts, the Helvetica font is reliably installed on Mac OS only, on Windows OS one should use Arial as a fallback. But there still exist some rendering flaws and other drawbacks when using Helvetica on Windows OS.

Helvetica is a sans-serif typeface by Max Miedinger and Eduard Hoffmann released in 1957.

Usage

In order to theme your personal Instapaper with Helvetapaper, you need to have Greasemonkey installed. Install Greasemonkey here.

Installing

Install Helvetapaper userscript

Uninstalling

To uninstall a Greasemonkey script from your browser, go to the Tools menu, select Greasemonkey, then Manage User Scripts, highlight the script and click Uninstall.
